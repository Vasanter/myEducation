# Виртуальное окружение нужно для изоляции проекта, чтобы все устанавливаемые библиотеки использовались только в рамках

Создание виртуального окружения происходит автоматически при создании проекта через IDE
Виртуальное окружение нужно для изоляции проекта, чтобы все устанавливаемые библиотеки использовались только в рамках
текущего проекта и не было конфликтов с другими зависимостями.

Первым делом создадим виртуальное окружение:
python3 -m venv venv # Mac OS / Linux
python -m venv venv # Windows

# активация => .venv\Scripts\activate
# деактивация => deactivate


СОЗДАНИЕ ВИРТУАЛЬНОГО ОКРУЖЕНИЯ С ОПРЕДЕЛЕННОЙ ВЕРСИЕЙ PYTHON!
py -3.14 -m venv env --создастся папка env с указанной версией python
