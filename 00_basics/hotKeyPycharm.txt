Горячие клавиши PyCharm по умолчанию (для Windows/Linux):

## **Навигация и редактирование**
| Горячие клавиши | Описание |
|-----------------|-----------|
| `Ctrl + Space` | Базовое автодополнение кода |
| `Ctrl + Shift + Space` | Умное автодополнение (учитывает контекст) |
| `Ctrl + P` | Показать параметры метода (сигнатуру) |
| `Ctrl + Q` | Быстрая документация |
| `Ctrl + F1` | Показать описание ошибки/предупреждения |
| `Ctrl + /` | Закомментировать/раскомментировать строку |
| `Ctrl + Shift + /` | Закомментировать блок кода |
| `Tab` / `Shift + Tab` | Увеличить/уменьшить отступ |
| `Ctrl + Alt + L` | Форматировать код по стандарту |
| `Ctrl + D` | Дублировать текущую строку или выделение |
| `Ctrl + Shift + ↑/↓` | Переместить строку/блок вверх/вниз |


## **Поиск и замена**
| Горячие клавиши | Описание |
|-----------------|-----------|
| `Ctrl + F` | Поиск в текущем файле |
| `Ctrl + R` | Замена в текущем файле |
| `Ctrl + Shift + F` | Поиск по всему проекту |
| `Ctrl + Shift + R` | Замена по всему проекту |
| `Ctrl + N` | Найти класс по имени |
| `Ctrl + Shift + N` | Найти файл по имени |
| `Ctrl + E` | Последние открытые файлы |
| `Alt + F7` | Найти использование элемента в проекте |


## **Рефакторинг**
| Горячие клавиши | Описание |
|-----------------|-----------|
| `Shift + F6` | Переименовать (переменную, метод, класс) |
| `Ctrl + Alt + M` | Выделить метод |
| `Ctrl + Alt + V` | Выделить переменную |
| `Ctrl + Alt + F` | Выделить поле |
| `Ctrl + Alt + C` | Выделить константу |
| `F6` | Переместить класс/файл |
| `Ctrl + Alt + O` | Оптимизировать импорты |


## **Выполнение и отладка**
| Горячие клавиши | Описание |
|-----------------|-----------|
| `Shift + F10` | Запустить текущую конфигурацию |
| `Shift + F9` | Запустить отладку |
| `F8` | Шаг через (Step Over) при отладке |
| `F7` | Шаг в (Step Into) при отладке |
| `Shift + F8` | Шаг из (Step Out) при отладке |
| `F9` | Продолжить выполнение (Resume) |
| `Ctrl + F2` | Остановить выполнение |


## **Работа с окнами и вкладками**
| Горячие клавиши | Описание |
|-----------------|-----------|
| `Ctrl + F4` | Закрыть активную вкладку |
| `Ctrl + Tab` | Переключение между открытыми файлами |
| `Alt + 1` | Показать/скрыть панель проекта |
| `Alt + 2` | Показать/скрыть панель Favorites |
| `Alt + 6` | Показать/скрыть панель TODO |
| `Alt + 9` | Показать/скрыть панель Version Control |
| `Esc` | Перейти в редактор кода (фокус) |
| `Shift + Esc` | Скрыть активную панель |


## **Прочие полезные команды**
| Горячие клавиши | Описание |
|-----------------|-----------|
| `Alt + Enter` | Показать контекстные действия (исправить ошибку) |
| `Ctrl + Alt + S` | Открыть настройки (Settings) |
| `Ctrl + Shift + A` | Найти действие (Action) |
| `Ctrl + Shift + F12` | Скрыть/показать все панели (максимизировать редактор) |
| `Double Shift` | Поиск везде (файлы, классы, действия, настройки) |
